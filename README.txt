kCalculator
----------------------------------

Description
kCalculator is a personal tool to help the users with their health. On the website, you can find out your daily nutrient goal and monitor the amounts of those you have entered.

----------------------------------

Technologies Used
HTML5
CSS3
JavaScript
Flask (backend)

----------------------------------

File Structure
/static: Static files like CSS and JavaScript
/templates: HTML pages
app.py: Main backend file

----------------------------------

Features
Calculate the nutrients daily goal for the user
User can add food items with specific nutritional values
Automatically calculate the total nutrients consumed
Display a detailed history of user entries
